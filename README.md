Working with template:
-----
Take a loot at [Dream-Otchlan](https://github.com/DreamPoland/dream-otchlan), example project based on platform.

**Required:**
-----
- Spigot-api 1.8.8, 1.12.2, 1.16.5, 1.17.1, 1.18.2, 1.19.3, 1.19.4, 1.20.+
- Java. (jdk version 11+, gradle target java8)

Working with NMS:
-----
If you don't have to use NMS, remove `mcversion` package.
